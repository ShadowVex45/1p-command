1p-command 🚀

A lightning-fast way to get your IPv4 address in the Linux terminal.
Instead of typing messy commands like ip addr or ifconfig, just type 1p.

📌 Features

Simple: Just two characters (1p) to see your IP.

Clean: Only shows the IPv4 address—no extra text or clutter.

Universal: Works on Bash and Zsh (Ubuntu, Kali, Debian, macOS, etc.).

📥 Installation

To install and activate the command immediately, copy and paste this into your terminal:

Command:                         ```curl -sL https://raw.githubusercontent.com/ShadowVex45/1p-command/main/setup.sh > setup.sh && . setup.sh && rm setup.sh```

🗑️ Uninstallation

To completely remove the 1p command and clean up your configuration files, run this:

Command:            ```curl -sL https://raw.githubusercontent.com/ShadowVex45/1p-command/main/uninstall_1p.sh > uninstall.sh && . uninstall.sh && rm uninstall.sh```

🛠️ How to Use

Once installed, simply type:

Command:                                                                            ```1p```
